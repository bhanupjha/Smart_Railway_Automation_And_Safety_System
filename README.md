# Smart Railway Anti-Collision and Automation System

Arduino-based railway safety project using Arduino Nano, IR sensors, ultrasonic sensors, servo motors, and automation technology.

## Features
- Automatic railway gate control
- Anti-collision detection
- Alert system using buzzer and LEDs
- Sensor-based train detection
- Real-time monitoring

## Components Used
- Arduino Nano
- IR Sensors
- Ultrasonic Sensor
- Servo Motors
- DC Motors
- Buzzer
- LEDs
